#  Fonksiyonel Web Hesap Makinesi

Bu proje, Python ve temel web teknolojileri (HTML/CSS/JS) kullanılarak geliştirilmiş interaktif bir hesap makinesi uygulamasıdır. 

# Proje Hakkında
Bu çalışma, "Efektif Prompt" ve "Fonksiyonel Programlama" mantığı kullanılarak modüler bir yapıda inşa edilmiştir. Sadece statik bir arayüz değil, arka planda çalışan matematiksel algoritmalar barındırır.

# Kullanılan Teknolojiler
* **Python:** Arka plan mantığı, modüler fonksiyonlar ve Exception Handling (hata yönetimi) için. (`hesap_makinesi.py`)
* **HTML5:** Sayfa iskeleti ve erişilebilirlik. (`index.html`)
* **CSS3:** Modern "Card UI" tasarımı, Grid yapısı ve görsel geri bildirimler (hover vb.). (`style.css`)
* **JavaScript:** Web arayüzünün interaktif hale getirilmesi.

# Öne Çıkan Özellikler
* Modüler fonksiyon yapısı.
* `try-except` blokları ile kullanıcı hatalarına karşı koruma.
* Sıfıra bölünme (ZeroDivisionError) kontrolleri.
* Temiz kod (Clean Code) prensipleri ve açıklayıcı yorum satırları.
